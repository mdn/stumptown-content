# intrinsicsize

This attribute tells the browser to ignore the actual intrinsic size of the image and pretend itâ€™s the size specified in the attribute. Specifically, the image would raster at these dimensions and `naturalWidth`/`naturalHeight` on images would return the values specified in this attribute.
