## Short description

The **HTML `<dd>` element** provides the details about or the
definition of the preceding term
([`<dt>`](/en-US/docs/Web/HTML/Element/dt))
in a description list
([`<dl>`](/en-US/docs/Web/HTML/Element/dl)).
