# `scrolling`

Indicates when the browser should provide a scrollbar for the frame.

## Values

### `auto`

Only when the frame\'s content is larger than its dimensions.

### `yes`

Always show a scrollbar.

### `no`

Never show a scrollbar.

## Type

String
